API

User ID and API Token of your Habitica account can be found in the settings under API section.

(?)
- Enabled: User ID and API Token are stored locally in your browser in encrypted state. Secure, but requires to enter Habitica login credentials in each browser. - Disabled: User ID and API Token are stored in Trello storage bound to your account. Less secure, but preserves Habitica login credentials even when switch between browsers.
